# WebInteraction

Automate web browsing, clicking, and validation with Python and Selenium."

## Features

- Open websites (http://qxf2.com/selenium-tutorial-main)
- Fill out forms
- Click buttons
- Hover over elements
- Select options
- Check text
- Search and find elements using:
    - CSS Selectors
    - XPath Locators

## Requirements

- Python 3.12.4
- Selenium 4.23.1
